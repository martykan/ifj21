dynstr_t * dynstr_append ( dynstr_t *  dynstr,
char  c

Appends a character to a dynamic string. Adds a character at the end of the string buffer. Increments len. Automatically appends a terminating null char after the new character. If necessary, reallocates the string buffer to fit the new character.

Parameters
dynstrPointer to an existing dynstr struct.
cCharacter to append.
Returns
The original dynstr pointer. NULL if failed.
