Appends a character to a dynamic string.

Adds a character at the end of the string buffer. Increments len. Automatically appends a terminating null char after the new character. If necessary, reallocates the string buffer to fit the new character.

Parameters
dynstrPointer to an existing dynstr struct.
cCharacter to append.
Returns
The original dynstr pointer. NULL if failed.

Converts a dynstr_t string value to a floating point value.

Uses strtod to convert the string stored in the passed dynstr_t into a double value. If the string doesn't represent a valid double, sets the global error flag.

Returns
Double value converted from passed string, or -1.0 if no conversion was possible.
