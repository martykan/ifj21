Detailed Description

Symbol table API.

Author
Patrik Korytar
Tomas Martykan
Filip Stolfa

DESCRIPTION

Symbol table interface for interaction. Used by other components of compiler for storing identifiers.

IMPLEMENTATION

Symbol table is composed of hash tables for global scope and local scopes. Local scope tables are inside a stack, the topmost of which represents most nested scope.
