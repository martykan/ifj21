Checks if the passed dynstr_t contains a valid ifj21 keyword. If it does, it calculates the corresponding token_type_t. Otherwise, the tested string is an identifier.

◆ scanner_init()

void scanner_init ( )

Initializes scanner for use.

Before it can be used, scanner needs its dynstr global variable set. That variable is used as a buffer for the string that has been processed so far. To avoid allocating and deallocating the buffer every time scanner is called, it is stored as a global variable.
